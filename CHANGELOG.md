# Історія змін

## 1.1.0 — 2026-07-18

- Додано синхронізацію до 10 локальних папок.
- Кожна локальна папка тепер має окрему однойменну папку всередині папки комп’ютера на Google Drive.
- Стару єдину папку й локальний стан синхронізації автоматично перенесено до нового формату.
- Додано список папок, відкриття та безпечне вилучення папки з резервного копіювання.
- Іконку системного трея замінено на вбудований логотип Lar’s Cloud.

## 1.0.0 — 2026-07-14

- Перший повний Windows-реліз.
- Google OAuth 2.0 із PKCE та DPAPI-сховищем токена.
- Інкрементальне резервне копіювання на Google Drive.
- SQLite-стан, resumable upload, графік, tray, автозапуск і сповіщення.
- Темний футуристичний WPF-інтерфейс.
- GitHub Releases updater із SHA-256.
- Self-contained build та Inno Setup Installer.
